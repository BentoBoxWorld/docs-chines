# BentoBox

BentoBox 是一个强大的 Bukkit 库插件，为岛屿风格的游戏如 SkyBlock, AcidIsland, OneBlock 等提供核心功能。它提供了如岛屿创建、团队管理、保护等功能。这些游戏和功能通过像迷你插件一样的插件被添加到 BentoBox。管理员可以自定义他们的服务器，拥有他们想要的确切游戏和功能。
