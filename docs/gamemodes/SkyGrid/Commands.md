# SkyGrid 命令

| 命令                  | 描述                               | 权限                      |
|-------------------|----------------------------------|-------------------------|
| **/skygrid**      | 开始 SkyGrid 游戏或传送到你的 SkyGrid 家园     | skygrid.skygrid        |
| **/skygrid go**   | 回家                               | skygrid.island.home    |
| **/skygrid language** | 选择语言                            | skygrid.island.language |
| **/skygrid sethome**  | 设置你的家传送点                         | skygrid.island.sethome |
