## 限制 (addon.yml)

将[gamemode]替换为游戏模式的名称，例如bskyblock, acidiland等。

| 权限                                  | 启用对于 | 描述                   |
|--------------------------------------|---------|-----------------------|
| [gamemode].mod.bypass                | op      | 玩家可以绕过限制        |
| [gamemode].limits.admin.limits       | op      | 玩家可以使用管理员限制命令 |
| [gamemode].limits.player.limits      | true    | 玩家可以使用限制命令     |
